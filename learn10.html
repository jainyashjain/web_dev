HTML Entities

HTML entities are used to display reserved characters or special symbols in HTML.

For display the html symbols on the screen we want to write a &symbol_code of symbols

Common HTML Entities

Entity Name Entity Code Displayed Character Description
& &amp; & Ampersand
< &lt; < Less than
> &gt; > Greater than
" &quot; " Double quote
' &apos; ' Single quote
© &copy; © Copyright symbol
® &reg; ® Registered trademark
$ &dollar; $ Dollar sign
😀 &#x1F600; 😀 Grinning Face Emoji

Example Usage

Here is how you can use entities:

To display a quote: "Hello, World!"

To include special characters: <div>Content</div>
